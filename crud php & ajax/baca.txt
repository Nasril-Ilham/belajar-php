Rekomendasi: Gunakan require_once

* require_once "koneksi.php"; *

require vs. include: Jika file yang disertakan hilang, include 
akan mengeluarkan peringatan (warning)
dan melanjutkan eksekusi skrip, sedangkan require akan
mengeluarkan kesalahan fatal (fatal error) dan menghentikan skrip. 
Untuk koneksi database, kegagalan koneksi adalah fatal, jadi require lebih aman.

_once: Memastikan file konfigurasi hanya disertakan satu kali 
dalam satu permintaan halaman, mencegah kesalahan pendefinisian ulang variabel.

DATATABLE ADALAH LIBRARY DARI JS

Fungsi DataTable()
.DataTable() adalah fungsi utama dari library DataTables. Ketika dipanggil pada sebuah tabel HTML,
 fungsi ini akan secara otomatis menambahkan berbagai fitur interaktif tanpa Anda perlu menulis kode HTML/CSS tambahan, termasuk:

Search Bar (Kolom Pencarian Global): Ini memungkinkan pengguna mencari data di seluruh kolom tabel.

Pagination (Penomoran Halaman): Membagi data menjadi beberapa halaman (misalnya, 10 baris per halaman).

Ordering/Sorting (Pengurutan): Mengizinkan pengguna mengurutkan data dengan mengklik header kolom.

Filtering/Entries (Filter Jumlah Data): Kotak dropdown untuk memilih berapa banyak baris yang ingin ditampilkan (misalnya, Show 10, 25, 50 entries).

Jadi, search bar dan fitur lainnya tidak muncul secara tiba-tiba, melainkan otomatis disuntikkan oleh library 
JavaScript DataTables sebagai bagian dari fungsinya untuk mengubah tabel statis menjadi tabel interaktif yang canggih (enhanced table).

Peran Sentral Library dalam Proyek Anda
Dalam proyek CRUD (Create, Read, Update, Delete) Anda, ada tiga library utama yang saling mendukung:

1. ðŸ¥‡ JQuery (Library Dasar)
Peran: JQuery adalah library JavaScript inti yang menyederhanakan manipulasi HTML (DOM), animasi, dan, yang paling penting bagi Anda, 
membuat permintaan AJAX (.load()).

Tanpa JQuery: Semua kode AJAX dan manipulasi elemen Anda akan jauh lebih panjang dan rumit.

2. ðŸ’… Bootstrap (Library Styling)
Peran: Bootstrap adalah framework CSS/JS yang menyediakan tampilan dan nuansa (estetika) untuk antarmuka Anda. 
Ia menyediakan gaya instan untuk tombol, formulir, navigasi, dan tata letak tabel.

Tanpa Bootstrap: Tabel Anda akan terlihat kuno dan polos, dan Anda harus menulis semua CSS tata letak sendiri.

3. ðŸ§  DataTables (Library Fungsionalitas Tabel)
Peran: DataTables adalah library JavaScript spesialis yang mengubah tabel HTML statis menjadi alat data interaktif.
Ia menyediakan "otak" untuk search bar, sorting, dan pagination.

Tanpa DataTables: Anda harus menulis script kompleks untuk mengimplementasikan pagination dan fitur pencarian secara manual.
