peredaan id daN class

id = biasanya di pake untuk memanggil logic 
tujuan utama : Identifikasi tunggal, JavaScript, linking.
Prioritas CSS : Sangat Tinggi

class = untuk styling dengan css atau tailwind 
tujuan utama : Styling (CSS), pengelompokan.
Prioritas CSS : normal

Aturan Emas

Selalu gunakan class untuk styling dan grouping. Gunakan id hanya jika 
Anda benar-benar perlu menandai elemen
itu secara unik untuk diakses oleh JavaScript atau sebagai target link (fragment URL).
